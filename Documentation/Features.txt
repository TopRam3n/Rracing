## 🏎️ 3. Garage / Builds System

**Garage Concept**

* Each user has one garage → contains multiple builds.
* **Build types:** Owned | Dream | For Sale.
* Builds can be **Public or Private (draft)**.

**Build Page (`/build/[id]`)**

* Cover image + gallery (lightbox).
* Specs summary: Make | Model | Year | Trim | Engine.
* Mods list (grouped by category).
* Power goal / Budget / Stage badges.
* Owner profile card + socials.
* Like button (♥) with real-time count.
* Share link + social preview.
* Build logs timeline (v2).

**Create Build Wizard**

1. Base Car → Make / Model / Year / Trim (+ photo).
2. Type  |  Stage  |  Power goal  |  Privacy toggle (Public/Private).
3. Mods list (category, brand, model, price, notes).

**Garage Directory (`/garage`)**

* Grid of all public builds.
* Filters: Make, Model, Type, Stage, Parish.
* Sort by Newest | Most Liked | Power | Budget.
* Infinite scroll / pagination.
* Cards show: cover image, name, likes, badges.
* “Clone as Dream Build” (v2).

**Like System**

* Logged-in users can like builds (1 per build).
* Likes stored in `likes` table + incremented counter in `builds.likes_count`.
* Top 5 most-liked → Landing Page.

**Privacy Rules**

* Public builds → visible to everyone.
* Private builds → visible only to owner.
* RLS enforced in Supabase.
